Constructs instance of Template class and serves as container for exposed properties and methods.

Histone(template[, baseURI]);

Parameters

  • template
    Template passed as String or AST.

  • baseURI
    Base URI of the template, which is passed as the first argument.

Return value

Returns instance of Template class.

Static properties

Histone.version

Version of Histone template engine as JavaScript String.

Histone.R_JS

Used in Template.render, Histone.register and Histone.invoke to set desired return or argument type.

Histone.R_HISTONE

Used in Template.render, Histone.register and Histone.invoke to set desired return or argument type.

Histone.R_STRING

Used in Template.render to set desired return type.

Histone.R_SELF

Used in Histone.register to specify that method returns self.

Histone.M_GET

Used in Histone.register and Histone.invoke to specify or invoke property getter.

Histone.M_CALL

Used in Histone.register and Histone.invoke to specify or invoke call handler.

Histone.M_TOJS

Used in Histone.register, Histone.invoke and Histone.toJavaScript to specify or invoke method which is responsible for converting Histone value into JavaScript value.

Base

Holds reference to the Histone.Base constructor (used as virtual parent for all Histone data types).

Undefined

Holds reference to the Histone.Undefined constructor.

Null

Holds reference to the Histone.Null constructor.

Boolean

Holds reference to the Histone.Boolean constructor.

Number

Holds reference to the Histone.Number constructor.

Histone.String

Holds reference to the Histone.String constructor.

RegExp

Holds reference to the Histone.RegExp constructor.

Macro

Holds reference to the Histone.Macro constructor.

Array

Holds reference to the Histone.Array constructor.

Date

Holds reference to the Histone.Date constructor.

Global

Holds reference to the Histone.Global constructor.

Histone.global

Holds reference to the instance of Histone.Global, which is used as global object in the template runtime.

Static methods

Histone.setCache

Enables or disables internal Histone cache

Histone.clearCache

Clears internal Histone cache.

Histone.setLanguage

Sets current locale

Histone.setResourceLoader

Sets up resource loader which is going to be responsible for handling network requests.

Histone.toJSON

Converts it's argument into a JSON String, calls previously registered toJSON method internally. Provides more convenient way of doing the same thing, rather than using Histone.invoke.

Histone.toString

Converts it's argument into a JavaScript String, calls previously registered toString method internally. Provides more convenient way of doing the same thing, rather than using Histone.invoke.

Histone.toNumber

Converts it's argument into a JavaScript Number, calls previously registered toNumber method internally. Provides more convenient way of doing the same thing, rather than using Histone.invoke.

Histone.toBoolean

Converts it's argument into a JavaScript Boolean, calls previously registered toBoolean method internally. Provides more convenient way of doing the same thing, rather than using Histone.invoke.

Histone.toJavaScript

Converts it's argument into a JavaScript value, calls previously registered Histone.M_TOJS method internally. Provides more convenient way of doing the same thing, rather than using Histone.invoke.

Histone.toHistone

Converts it's argument into a Histone value. If the value is previously registered value, returns the value itself, otherwise attempts to convert passed value.

Histone.loadText

Loads text file using resource loader, previously specified using Histone.setResourceLoader asynchronously.

Histone.loadJSON

Loads JSON file using resource loader, previously specified using Histone.setResourceLoader asynchronously.

Histone.require

Loads and processes the template file using resource loader specified using Histone.setResourceLoader asynchronously.

Histone.register

Registers virtual method that adds or modifies one of the built-in Histone data types or custom data type.

Histone.unregister

Unregisters (removes) previously registered (by Histone.register) virtual method.

Histone.invoke

Calls previously registered (by Histone.register) virtual method and returns the result of it's invocation.

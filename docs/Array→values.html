Array→values

Jump to bottom
Ruslan edited this page on Mar 20, 2015 · 6 revisions

Returns all the values from the array and indexes the array numerically.

Array->values()

Basic usage:

{{["size": "XL", "color": "gold"]->values->toJSON}} = ["XL","gold"]

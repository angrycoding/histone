Histone.loadJSON

Jump to bottom
Ruslan edited this page on Jul 12, 2016 · 2 revisions

Loads JSON file using resource loader, previously specified using Histone.setResourceLoader asynchronously.

Histone.loadJSON(uri, callback, requestProps);

Parameters

  • uri
  • callback
  • requestProps

Return value

Method returns no value.

Basic example

// ...
Histone.loadJSON('file.json', function(result) {
    console.info(result);
});
